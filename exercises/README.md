# Ejercicios Prácticos

Este directorio contiene ejercicios prácticos organizados por capítulo.

## Estructura

Cada carpeta de capítulo contiene:
- Notebooks de Jupyter con ejercicios
- Scripts de Python
- Datos de ejemplo (si son pequeños)
- README con instrucciones específicas

## Cómo usar

1. Navega al capítulo correspondiente
2. Lee el README del capítulo
3. Completa los ejercicios en orden
4. Marca como completado en el README principal
